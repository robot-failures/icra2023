Topics of interest

The topics of interest for the workshop include, but are not limited to, the following:
  • Robot failure modelling approaches
  • Techniques for robot execution monitoring, avoidance, and recovery
  • Approaches for preventing and handling failure propagation, particularly in mobile manipulation
  • Approaches for robot fault diagnosis
  • Causal learning for failure understanding
  • Failure-aware learning
  • Failure explainability
  • Failure management in human-robot collaboration
  • Datasets that include execution and/or interaction failures
  • Robot verification and/or testing approaches that consider failure scenarios

Submission guidelines

Submissions to the workshop can be either short papers (4 pages) or conference-length papers (6 pages). Submissions describing work in progress that authors would like to discuss with the workshop participants are encouraged and welcome. The workshop does not have official proceedings, but the submitted papers will be posted on the workshop website if the authors agree.

Each submitted paper will receive at least two reviews, based on which an acceptance decision will be made. Papers should be submitted via EasyChair. Please note that email submissions will not be accepted.
